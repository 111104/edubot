# EduBot — AI оқу ассистенті
### Astana IT University колледжі

EduBot — колледж студенттеріне арналған жасанды интеллект негізіндегі оқу ассистенті. КМ03, КМ04, КМ05, КМ06, КМ07 модульдері бойынша студенттердің сұрақтарына жауап береді, тьютор рөлін атқарады.

## Мүмкіндіктері
- 5 модуль бойынша AI жауаптары
- Қазақ / орыс / ағылшын тілдерінде жұмыс
- Тьютор педагогикасы — дайын жауап бермей, бағыттайды
- Жеңіл интерфейс

## Іске қосу
1. Репозиторийді клондаңыз
2. `index.html` файлын ашыңыз
3. Anthropic API кілтін енгізіңіз
4. Дайын!

## GitHub Pages арқылы жариялау
